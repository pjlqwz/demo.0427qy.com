泰国还有那么一座岛叫大长岛，那么的美，你知

2019-01-07 14:40

一直隐藏着的大长岛，很多人都没听过这个名字。但看过《机械师2》的朋友不知道会不会有印象，杰森斯坦森去找杨紫琼时，虽然官宣说是丽贝岛，但实际抵达的那个离岛就是大长岛喔！ 趁现在这里还没有被过度开发，没人人人人人人海，赶紧来吧！

 

大长岛的大概位于普吉岛西侧和甲米府东侧之间的攀牙湾海域。也是普吉附近最大的离岛了，被誉为安达曼海上最后一片“世外桃源”。

 

它是因为狭长的外形而得名的，中文名又有音译为：瑶亚岛。岛上常住人口约8000人，

众所周知泰国是个有信仰的国度，但大长岛上特别的是，岛上居民大多数却信奉的是伊斯兰教。

 

为什么要来大长岛

 

因为这里还没有被完全的开发，所以完全避开了熙熙攘攘的旅游团队大军，还了岛上一片清净。相较于大长岛旁边的皮皮岛来说，这里可以说是人烟稀少啊~

 

而且前往大长岛的距离又比皮皮岛近多了~

 

在这个静瑟的小岛上，找家度假村，住上几天，悠闲放空。远离乌泱泱的人群，独享一片纯净的碧海蓝天，这不才是度假的精髓么！

 

大长岛的最佳季节

 

泰国属于热带季风气候。 一年三季，分别是热季（2月中旬至5月中旬）、雨季（5月下旬至10月中旬）和凉季（11月至次年2月中旬）。

 

所以大长岛这边就建议是凉季来玩，天气最好了！毕竟海岛的风景美，都是靠美好的天气做衬托的~

大长岛怎么去

 

岛上的酒店也不多，基本都是非常赞的五星级酒店（按国内标准衡量的话…）。不过因为不是开发完善的岛屿，所以岛上住宿的性价比是非常高的！

 

类比五星级酒店的服务，一般酒店都会有自己的接送服务，只不过酒店的收费都会相对高一些。

 

所以我在这里主要推荐乘坐公共交通工具的话，要怎么去呢？

 

从普吉、甲米和攀牙出发的话，往返的船只分为渡轮和快艇两种。渡轮是比较大的船，船速缓而稳，船底可以停车；快艇就航行速度快，比较颠簸。

 

例如：渡轮从普吉的到大长岛时长大概1小时，快艇约30分钟左右可以到达。

 

Bang Rong 码头：为长尾船和快艇拼船两种，船次从早上7.50开始到下午17.30，约半小时发一次船。

快艇出发时间为：07:40 / 08:40 / 09:15 / 09:40 / 10:30 / 11:30 / 12:30 / 13:15 / 14:30 / 15:00 / 16:00 / 17:00 / 17:30 。

 

TIP.个人建议你可以坐快艇，比较快，而且汽油味没有那么重，坐在里面也不会太晒。
